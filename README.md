# Dashboard de Catálogo de Productos

Dashboard interactivo para la exploración, visualización e interpretación de un
catálogo de productos de comercio electrónico, desarrollado con **React +
TypeScript + Vite + MUI + MUI-X** y conectado a **Firebase Realtime Database**.

## Objetivo

Permitir a un analista comercial explorar precios, categorías y calificaciones
de productos mediante indicadores (KPIs), un gráfico de precio promedio por
categoría, una tabla detallada y filtros interactivos.

## Dataset

Retail Product Catalog Dataset (Kaggle):
https://www.kaggle.com/datasets/noopurbhatt/retail-product-catalog-dataset

## Tecnologías

- React 19 + TypeScript + Vite
- MUI (Material UI) y MUI-X (Data Grid y Charts)
- Firebase Realtime Database
- Despliegue en GitHub Pages con gh-pages

## Ejecución local

```bash
npm install
npm run dev
```

## Despliegue

```bash
npm run deploy
```
